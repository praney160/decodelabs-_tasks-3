# 🛡️ DecodeLabs Internship - Project 3

## Phishing Awareness Analysis

### 📌 Overview

This project analyzes email messages to identify potential phishing attacks by detecting suspicious keywords and links commonly used in social engineering attacks.

---

## 🎯 Objective

- Detect phishing indicators in email messages.
- Identify suspicious keywords and links.
- Explain why the message is unsafe.
- Promote cybersecurity awareness.

---

## ✨ Features

- Detects phishing keywords.
- Identifies suspicious URLs.
- Lists security red flags.
- Displays phishing analysis results.
- Easy-to-use command-line interface.

---

## 🛠️ Technologies Used

- Python 3
- Visual Studio Code
